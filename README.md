## tensorflowjs-anguler-webmnist

## Setting
* node : 12.18.2
* Angular CLI : 10.0.1
* tensorflow/tfjs : 2.0.1
* chart.js : 2.9.3
* ng2-charts : 2.3.2
* webgl2 : 0.0.5

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
